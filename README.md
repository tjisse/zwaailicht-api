# zwaailicht-api

Tiny RESTful API for controlling an alarm light connected to a serial port.

## Prerequisites

You will need [Leiningen][] 2.0.0 or above installed.

[leiningen]: https://github.com/technomancy/leiningen

## Running

To start the application, run:

    lein ring server
